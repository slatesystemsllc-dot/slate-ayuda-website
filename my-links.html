Una página de acceso rápido con todos tus links importantes en un solo lugar.

¿Qué Es Mis Links?

Tu sitio web tiene una página especial en tusitio.com/my-links que te da acceso rápido a:

🌐

Tu Sitio Web

Link rápido para compartir tu sitio web con clientes o en redes sociales.

📱

Tu Teléfono de Negocio

Toca para copiar tu número de negocio al portapapeles.

⭐

Pedir Reseñas

Abre el formulario de marketing para agregar clientes felices. También tiene un boton Copy Link para guardarlo.

📍

Google Business

Acceso rápido para manejar tu perfil de Google Business. ¿Necesitas ayuda? Ve nuestra Guía de Google Business.

Descargar App

Links para descargar la app Lead Connector para iPhone y Android.

Como Usarla

  1. Guardala en favoritos : Guarda tusitio.com/my-links en la pantalla de inicio de tu teléfono para acceso rápido
  2. Comparte tu sitio web : Toca la tarjeta del sitio web para abrirlo, luego comparte el link con clientes
  3. Copia tu teléfono : Toca la tarjeta del teléfono para copiar tu número de negocio
  4. Agrega clientes para reseñas : Toca "Request a Review" para abrir el formulario, o toca "Copy Link" para guardar el link

Pro Tip

Agrega la página /my-links a la pantalla de inicio de tu teléfono. En iPhone: Abre en Safari, Compartir, Agregar a Pantalla de Inicio. En Android: Abre en Chrome, Menu, Agregar a Pantalla de Inicio.

El Boton de Pedir Reseñas

Este es tu link más importante. Después de cada cliente (cada trabajo, sin importar cómo salió), usa esto para agregarlos al sistema de reseñas:

Lo que pide el formulario:

  • Nombre : Obligatorio
  • Apellido : Opcional pero ayuda
  • Teléfono : Obligatorio (para recordatorios por SMS)
  • Email : Opcional (para recordatorios por email)

Que pasa después de enviar:

  1. Recibes un texto de confirmación (en 1 minuto)
  2. Tu cliente recibe una solicitud de reseña amable
  3. Si no responden, reciben recordatorios automáticos
  4. Si el número no funciona, te notificamos

¿Problemas?

¿Links no funcionan?

Si algun link muestra error, escríbenos al +1 (831) 226-7831. Puede que necesitemos actualizar tus ajustes.

¿Página no carga?

Intenta limpiar el cache del navegador o abrir en una ventana privada/incógnito.

¿No encuentras tu página?

Tu página my-links está en: tusitio.com/my-links. Reemplaza "tusitio.com" con tu dominio real.

¿No se copia el teléfono?

Algunos navegadores bloquean el acceso al portapapeles. Intenta usar Safari (iPhone) o Chrome (Android).
